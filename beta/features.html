Virtual PTU's Features

Save Editor

Quickly create campaigns with base data or start from scratch.

Import resources like map images, sprites, tokens, sound tracks and more.

Import resources into the save file to use them as tokens, map props, maps, sound effects, music and more... Hover over a resource to get a preview (Sound preview support planned, but not available)

Create, edit or delete Pokémon species, moves, abilities and items.

Edit the Pokédex easily... Add, Edit, Delete or View anything from the Pokémon species, moves, items and abilities... Including Pokémon evolutions, move polls and more to support as much homebrew as possible

Character management for NPC's and players alike.

Easily organize Characters, NPC's, Pokemon and more in a folder like structure, with a basic view permissions overview similar to Roll20

Make and manage NPCs or your player's characters and Pokémon.

Easily edit status conditions, Stats, Evasion, Moves, EXP, Gender and more from a basic interface

Basic permissions system for your NPCs or your player's characters and Pokémon.

Easy permissions to keep your bosses, enemies and NPCs mysterious and secret. Give permission to view or edit an entity on a per-player basis.

Simple user authentication and permissions system, without using passwords.

We don't want to store a username and password on the GM's computer because of various security reasons... so we designed an authentication system that allows strong user management and a coming permissions system with a unique 32Character long key for each user

Custom Battle Typing.

Want to create a custom type? Well, now you can... You can create, edit or delete types and in a future version, have the option to define a custom effect modifier
